# 🎀 인스타그램 클론 코딩을 통한 포트폴리오 제작
- KDT-4기 3번째 미니 프로젝트(4인 그룹)

---
## 💻 Description

인스타그램의 개인 프로필 기능과 UI를 모방하여<br>
개인 포트폴리오를 HTML, CSS를 활용하여 제작하였습니다. <br><br>

- HTML, CSS를 활용한 웹 페이지 디자인
- Iframe을 활용하여 모바일 구성으로 화면 배치

---

- **실제 개발 기간:** 2023.07.27. ~ 2023.07.28.
